# tao-foobaz-fe

Dummy extension for testing npm releasing

## Install dependencies

`npm install`

## Build sources

`npm run build`

## Run tests

`npm run test`

## Lint code

`npm run lint`

## Test commits

1. fix: a small fix
2. feat: a medium feature
3. feat(yeah): a yeah-scoped feature
4. chore: do a task
5. yikes!: this will break
6. merge commit 1
7. merge commit 2
